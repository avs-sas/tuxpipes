usage: tuxpipes [-h] [-a] [-d PIPELINE_NAME] [-r OLD_NAME NEW_NAME] [-l [FILTER]] [-i] [-o [FILE_NAME]] [-y] [-n] [-c [FILE_NAME]] [input_string]

Positional Arguments

input_string

Input format: PIPE_ELEMENT[:PIPE_ELEMENT]*

Named Arguments

-a, --add

Add a new pipeline based on the input string

Default: False

-d, --delete

Delete a pipeline

-r, --rename

Rename a pipeline

-l, --list

List all pipelines

-i, --info

Show pipeline info based on the input string

Default: False

-o, --output

Create a bash script for running a pipeline

-y, --yes

Answer yes to all questions

Default: False

-n, --no

Answer no to all questions

Default: False

-c, --commands

Set a pre_commands.json file
